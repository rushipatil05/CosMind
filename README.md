## 🛠️ Installation & Setup

To run the project locally:

```bash
# 1. Clone the Repository
git clone https://github.com/DhanushNadar/Cosmind

cd Cosmind

# 2. Install Dependencies
npm install

# 3. Start Development Server
npm run dev
```
